# 说明： '井' 号后面部分没有效果，空格空行不影响结果，大小写不敏感。
# 奇数行为电影电视剧名，豆瓣中文英文名， '.'和 空格 不影响结果。
# 偶数行为imdb或则豆瓣链接，除了全连接(https?)，imdb 支持 tt1234567 形式。

# 示例：
# Einstein and Einstein
# https://movie.douban.com/subject/25716096/ 

# Cinderella.Chef                  # 萌妻食神
# https://douban.com/subject/30395527/    # 最好使用带movie形式链接

# CindeGreat.Journey.Of.Teenagers  # 少年歌行
# https://douban.com/subject/30345127/
