# 资源名中 '.'和 空格 不影响结果。
# 注意分隔符(+)个数，可少，可多，但是最后一个须为资源名。
# 0 或者 no 表示不发布，1 或则 yes 表示发布。

# 示例：
# whu + hudbt + npupt + nanyangpt + byrbt + cmct + tjupt + 资源名字(部分或全部，支持匹配）

# yes + yes + no + no + no + no + no + Douro.Mainland.E      # 斗罗大陆  仅发 whu 和 hudbt
